* Apresentação:

O sistema Construção Vendas foi desenvolvido para atender de forma prática e eficiente as necessidades de lojas de materiais de construção. Ele busca substituir controles manuais, cadernos e planilhas, centralizando as operações mais importantes da loja em um único ambiente digital.

Com o sistema, é possível cadastrar produtos, controlar o estoque, gerenciar clientes e registrar vendas de forma rápida e segura. Além disso, permite consultar o histórico de vendas, auxiliando na tomada de decisões.

A proposta é oferecer um sistema simples, organizado e eficiente, melhorando o atendimento ao cliente e aumentando a produtividade da loja.

Descrição do Projeto:

O sistema será composto pelas seguintes funcionalidades:
 * Cadastros:
Produtos
Clientes
Vendas
Estoque
 * Usuários e Permissões:
Gerente: acesso total ao sistema
Atendente: cadastro de produtos, clientes e vendas
* Requisitos Funcionais:
RF01 - Cadastro de Produtos:
Permitir cadastrar, editar e excluir produtos com nome, categoria, quantidade e preço.
RF02 - Controle de Estoque:
Atualizar automaticamente o estoque após cada venda.
RF03 - Cadastro de Clientes:
Registrar e consultar dados dos clientes.
RF04 - Registro de Vendas:
Registrar vendas com produtos, quantidade, valor e cliente.
* Requisitos Não Funcionais:
RNF01 - Usabilidade:
Interface simples e fácil de usar.
RNF02 - Desempenho:
Operações devem ocorrer em até 2 segundos.
RNF03 - Segurança:
Acesso protegido por login e senha.
RNF04 - Portabilidade:
Compatível com Windows e Linux.
  * Tecnologias Utilizadas:
Java
Programação Orientada a Objetos (POO)
MySQL
